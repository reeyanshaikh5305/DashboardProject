# DashboardProject
Data Science Salary Dashboard
Overview

This project is an interactive Microsoft Excel dashboard that provides insights into salaries for various data-related professions around the world.

Using Excel's data analysis and visualization features, the dashboard allows users to dynamically explore salary information based on:

Job Title
Country
Employment Type

The dashboard automatically updates charts and summary statistics whenever a filter is selected.
